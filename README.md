# Java-Lap-Exercises
java programming lap exercises developed using NetBeans

##  Projects 
This repository contains Java lab exercise projects completed. Each project focuses on applying object-oriented programming concepts.

###  1. Bank Account System
A simple banking system that allows users to create accounts, deposit money, withdraw money, and check account balances.

###  2. Maintenance Plan Car System
A system to manage car maintenance plans, including service schedules and maintenance details.

###  3. Student Record Management System
A system used to store and manage student information such as names, IDs, courses, and grades.

###  4. Parking System
A parking management system that tracks available parking spaces and vehicle entries and exits.

###  5. Product Counting System
A system that counts and manages products, including quantities and basic product information.

##  Technologies Used
- Java
- NetBeans IDE
- Object-Oriented Programming (OOP)
